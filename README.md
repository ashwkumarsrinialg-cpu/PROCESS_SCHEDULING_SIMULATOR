# Hybrid EEVDF+MLFQ: A Thermally-Aware, Hardware-Validated CPU Scheduler for Mobile Linux Platforms

**Preprint**: [paper.pdf](paper.pdf)  
**Date**: June 28, 2026

## Overview

This repository contains all artefacts accompanying the paper:

> "Hybrid EEVDF+MLFQ: A Thermally-Aware, Hardware-Validated CPU Scheduler for Mobile Linux Platforms"

The work introduces a hybrid scheduler that combines Multi-Level Feedback Queues (MLFQ) with Linux EEVDF, augmented with thermal scaling τ(T), urgency bias α, and big.LITTLE capacity normalisation.

## Repository Contents

- **`scheduler_sim.py`** — Hardware-informed discrete-event simulator (main simulation engine)
- **`final_mobile.zip`** — Complete archive containing:
  - Kernel prototype patch (`0001-sched-hybrid.patch`)
  - Android benchmark harness (`android_bench/`)
  - Sim-to-real validation scripts (`validation/`)
- **`plots/`** — All figures used in the paper (generated by `scheduler_sim.py`)
- **`paper.pdf`** — Final preprint
